- Product scope: retirement planning, household financial planning, and explainable simulations

## Workflow Contract

Read these files before AI-assisted implementation or review:

1. `AGENTS.md`
2. `REVIEW.md`
3. `.github/pull_request_template.md`
4. `.github/ai-automation.yml`
5. Relevant files under `docs/`

Use one issue, one branch, and one pull request at a time. Do not merge automatically.

## GitHub Project

The repository workflow targets user-owned Project #2 and synchronizes `Status`, `Priority`, `Risk Tier`, `AI Owner`, and `Area`. Projects V2 writes require the `PROJECTS_TOKEN` repository secret; the workflow fails fast when it is missing.
